Socialicious
============

A social media icons library in font form for Bootstrap.

Icons currently include:
- Facebook
- Twitter
- Bebo
- Bread
- Dribbble
- Flickr
- Foursquare
- Github
- Google Plus 
- Instagram
- Last.fm
- Linkedin
- Pinterest
- Klout
- Tumblr
- Vimeo
- Wordpress
- Vevo

More to come!
